AI Document Q&A System (RAG-based)

A production-grade, multi-user AI Document Q&A platform leveraging Retrieval Augmented Generation for accurate, grounded answers from uploaded documents.

  • PDF ingestion with text chunking (overlap) and FAISS-based vector search for semantic retrieval
  • Hybrid search combining semantic + keyword re-ranking to improve recall and reduce hallucinations
  • LLM inference via Groq with confidence scoring, strict grounding prompts, and source citations
